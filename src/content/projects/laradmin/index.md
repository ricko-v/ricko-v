## Deskripsi
Sebuah admin template dengan teknologi Laravel + Inertia + React

Proyek ini dibuat untuk mengimplementasikan belajar role management dengan menggunakan laravel spatie.

## Fitur

- Dashboard
- CRUD Role
- CRUD Permission
- CRUD User
- Role Management

## Teknologi
- Laravel 11
- Inertia v2
- Reactjs
- Shadcn UI
- Spatie
